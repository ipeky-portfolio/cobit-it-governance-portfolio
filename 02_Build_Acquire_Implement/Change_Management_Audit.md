# COBIT BAI06: Managed Changes

## Prozess-Checkliste:
1.  **BAI06.01:** Ist ein Change Advisory Board (CAB) definiert?
2.  **BAI06.02:** Werden Änderungen vor der Implementierung getestet?
3.  **BAI06.03:** Gibt es einen Rollback-Plan?

## Audit-Beispiel:
* **Feststellung:** Änderungen an der Firewall wurden direkt in der Produktion ohne vorherigen Test im Staging-System durchgeführt.
* **Auswirkung:** Netzwerkausfall für 2 Stunden.
* **Bewertung:** **Major NC** (Prozessreifegrad 2).
